Oyunu Başlat ➔

🖥️ İş Kompyuteri üçün Çoxnəfərli Oyun Serverinin Qurulması

İş kompyuterlərində admin icazələri olmadan və kart məlumatı daxil etmədən, tamamilə pulsuz və 24/7 işləyən çoxnəfərli oyun serverini (GitHub Codespaces ilə) necə qurmaq olar:

1

GitHub-da Codespace Yaradın

Öz layihənizə (github.com/ulviabbasli/Cs) daxil olun, yaşıl "<> Code" düyməsinə klikləyin, Codespaces tabına keçin və "Create codespace on main" düyməsini sıxın.

2

Server Komandasını İşə Salın

Açılan virtual mühitdə, ekranın aşağısındakı terminala bu komandanı yapışdırıb Enter sıxın:

docker run -d -p 27015:27015/udp -p 27016:27016 yohimik/cs-web-server
3

Port Görünüşünü Aktivləşdirin

Aşağı paneldə "Ports" (Portlar) tabına keçin, siyahıdakı 27016 portunun üzərinə sağ klikləyin, "Port Visibility" ➔ "Public" (İctimai) edin.

4

IP-ni Kopyalayın və İcazə Verin (MƏCBURİ)

  1. Portun qarşısındakı "Forwarded Address" linkini kopyalayın (məsələn, https://...-27016.app.github.dev/).
  2. Həmin linki brauzerinizdə yeni tabda açın və çıxan xəbərdarlıq səhifəsində yaşıl "Continue" düyməsini klikləyin.
  3. Linkdən https:// və sondakı maili xətt (/) işarələrini silərək təmiz server ünvanını əldə edin.
5

Qoşulun və Oynayın!

Təmizlənmiş server ünvanını (məsələn: ...-27016.app.github.dev) IP qutusuna yazın, "Qoşul və Başlat" klikləyin və oyun açılan kimi avtomatik serverə bağlanın!

ℹ️ Tez-tez Soruşulan Suallar (FAQ)

İş kompyuterində siçan (mouse) oyun zamanı niyə kənara qaçır?

Brauzer oyunlarında tam siçan nəzarəti üçün Pointer Lock aktiv olmalıdır. Oyuna başladıqdan sonra ekrana klikləyin və brauzerin siçanı kilidləməsinə icazə verin. Oyundan çıxmaq üçün ESC düyməsini sıxın.

Səs və mikrofon dəstəklənirmi?

Bəli! Brauzerdə mikrofon icazəsi verdikdən sonra oyundaxili səsli söhbət (voice chat) serverde işləyəcək.
