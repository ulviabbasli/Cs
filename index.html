🚀 GitHub Pages Yayım Təlimatı

Bu portalı öz GitHub səhifənizdə yayımlamaq və iş yoldaşlarınızla paylaşmaq üçün aşağıdakı addımları izləyin.

1

GitHub-da Yeni Anbar (Repository) Yaradın

GitHub profilinizdə (ulviabbasli) daxil olun və Cs adında yeni public (ictimai) bir anbar yaradın.

2

Faylları GitHub-a Yükləyin

Metod A: Brauzer vasitəsilə (Heç bir proqram yükləmədən - Ən sadə):

  1. GitHub-da yaratdığınız Cs anbarına (repository) daxil olun.
  2. "uploading an existing file" və ya "Add file" ➔ "Upload files" düyməsini klikləyin.
  3. Lokalda olan index.html, style.css, app.js və README.md fayllarını sürükləyib brauzerə buraxın.
  4. Səhifənin aşağısında "Commit changes" düyməsini sıxın.

Metod B: Git Komandaları ilə (Əgər kompyuterdə Git varsa):

git init
git add .
git commit -m "First commit: CS 1.6 Web Hub"
git branch -M main
git remote add origin https://github.com/ulviabbasli/Cs.git
git push -u origin main
3

GitHub Pages-i Aktivləşdirin

GitHub anbarınızın (Repository) Settings ➔ Pages bölməsinə daxil olun. Build and deployment hissəsində Branch seçimini main (və qovluğu /root) olaraq təyin edib Save düyməsini sıxın.

4

Paylaşın və Oynayın!

Bir neçə dəqiqə ərzində saytınız aktivləşəcək. İş yoldaşlarınız https://ulviabbasli.github.io/Cs/ linkinə daxil olaraq bu portalı aça biləcəklər!

ℹ️ Tez-tez Soruşulan Suallar (FAQ)

İş kompyuterində siçan (mouse) oyun zamanı niyə kənara qaçır?

Brauzer oyunlarında tam siçan nəzarəti üçün Pointer Lock aktiv olmalıdır. Oyuna başladıqdan sonra ekrana klikləyin və brauzerin siçanı kilidləməsinə icazə verin. Oyundan çıxmaq üçün ESC düyməsini sıxın.

Maksimum neçə nəfər oynaya bilərik?

Play-CS.com otaqları adətən 32 nəfərə qədər oyunçunu dəstəkləyir. Lokal ofis şəbəkənizdə limitsiz şəkildə komandalara bölünüb oynaya bilərsiniz.

Səs və mikrofon dəstəklənirmi?

Bəli! Brauzerdə mikrofon icazəsi verdikdən sonra oyundaxili səsli söhbət (voice chat) dəstəklənən otaqlarda işləyəcək.
