👾 WebXash3D CS 1.6 Client (Brauzerdə Birbaşa)

Bu bölmədə rəsmi WebGL/WebAssembly əsaslı CS 1.6 klienti yerləşdirilib. Oyun sizin brauzerinizdə, tamamilə lokal olaraq işə düşür.

Avtomatik Yükləmə: Layihəyə game_part_*.bin faylları yüklənibsə, aşağıdakı emulyatordan Counter-Strike 1.6 seçib Start Demo kliklədiyinizdə oyun avtomatik olaraq heç bir fayl yükləmədən başlayacaq!

🌐 İş Yoldaşları ilə Çoxnəfərli (Multiplayer) Oyun

Birlikdə oynamaq üçün komandanızdan bir nəfər lokal şəbəkədə (ofisdə) və ya internetdə CS Dedicated Server qurmalıdır. Digər oyunçular həmin serverin WebSocket IP adresini daxil edib oyuna daxil ola bilərlər.

🖥️ İş Yoldaşları ilə Oyun Qurmaq (Lokal Server Setup)

Əgər oyunçular eyni ofis şəbəkəsindədirsə (LAN), bir nəfər öz kompyuterində asanlıqla Counter-Strike Dedicated Server qura bilər və hamı eyni oyuna qoşula bilər:

1

Lokal Serverin Qurulması (İki Üsuldan Birini Seçin)

Üsul A: Heç bir proqram yükləmədən (Yalnız Windows PowerShell ilə - TÖVSİYƏ OLUNUR):

  1. Ofisdə server olacaq Windows kompyuterdə layihə qovluğuna daxil olun.
  2. run_local_server.ps1 faylının üzərinə sağ klikləyin və "Run with PowerShell" (PowerShell ilə işlət) seçin.
  3. Bu skript avtomatik olaraq Dedicated Serveri arxa planda işə salacaq və brauzerlərin qoşulması üçün WebSocket proxy-ni aktivləşdirəcək.

Üsul B: Əgər kompyuterdə Docker varsa:

docker run -d -p 27015:27015/udp -p 27016:27016 yohimik/cs-web-server
2

Server IP Ünvanının Tapılması

Serveri quran şəxs öz kompyuterinin lokal IP ünvanını tapmalıdır (PowerShell-də ipconfig komandası ilə, məsələn: 192.168.1.50).

3

Oyuna Qoşulma

Bütün iş yoldaşlarınız "BRAWSER KLİENT (WASM)" bölməsinə daxil olaraq IP sahəsinə server IP-sini və 27016 portunu daxil etməlidir:

192.168.1.50:27016

"Qoşul və Başlat" düyməsini kliklədikdən sonra oyuna daxil olun. Oyun başladıqdan sonra MULTIPLAYER ➔ LAN Games bölməsində qurulmuş serveri görəcəksiniz!

ℹ️ Tez-tez Soruşulan Suallar (FAQ)

İş kompyuterində siçan (mouse) oyun zamanı niyə kənara qaçır?

Brauzer oyunlarında tam siçan nəzarəti üçün Pointer Lock aktiv olmalıdır. Oyuna başladıqdan sonra ekrana klikləyin və brauzerin siçanı kilidləməsinə icazə verin. Oyundan çıxmaq üçün ESC düyməsini sıxın.

Maksimum neçə nəfər oynaya bilərik?

Play-CS.com otaqları adətən 32 nəfərə qədər oyunçunu dəstəkləyir. Lokal ofis şəbəkənizdə limitsiz şəkildə komandalara bölünüb oynaya bilərsiniz.

Səs və mikrofon dəstəklənirmi?

Bəli! Brauzerdə mikrofon icazəsi verdikdən sonra oyundaxili səsli söhbət (voice chat) dəstəklənən otaqlarda işləyəcək.
